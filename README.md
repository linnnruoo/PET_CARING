# CS2102 TEAM 35 TOPIC PET_CARING

## Quick Start
First, clone repo and login when prompted
```
$ git clone https://github.com/linnnruoo/PET_CARING.git
$ cd PET_CARING
```
To start the server on localhost:5000:
```
$ cd server
$ yarn
$ yarn start
```
To start the client on localhost:3000:
```
$ cd client
$ yarn
$ yarn start
```
